
# Getting Started

1. Get API Key

[GROQ API Key](https://console.groq.com/keys)

2. Edit `GROQ_API_KEY` environment variable.

2. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
